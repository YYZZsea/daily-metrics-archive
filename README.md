# Daily Metrics Archive

Private dated reporting snapshots.

This repository contains dated complete ranking CSV files, wide-format ranking timeline CSV files, and top-100 HTML technical trend galleries for Prism and Early Signal across SPX, Russell 2000, Nasdaq 200, and Nasdaq Liquid 500.

Complete ranking CSV files are retained in each dated folder for archival purposes but are intentionally omitted from the generated web index. The web index links only to ranking timelines and trend galleries.

Source code, skills, raw market data, Markdown reports, Excel workbooks, logs, and account information are intentionally excluded. Each timeline CSV keeps one stock per row, newest dates first, and three columns per date: predicted rank, close, and next-snapshot return.

> Research and educational use only. Not financial advice.
